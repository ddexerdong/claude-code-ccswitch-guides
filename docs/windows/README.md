## Windows Notes

- Windows users should pay attention to PowerShell / CMD / Git Bash differences.
- Git for Windows is recommended because it provides both Git and Git Bash.
- If `claude` or `git` is not recognized, reopen the terminal before deeper troubleshooting.
- Do not expose API keys when sharing screenshots or terminal output.
