## 使用建议

1. 第一次部署时，先看安装配置指南。
2. 已经部署完成后，优先使用维护指南做自检和排错。
3. 所有 API Key、Token、Bearer Token 和敏感请求头都不要公开。
4. 模型名、Base URL、安装方式以官方文档为准。
