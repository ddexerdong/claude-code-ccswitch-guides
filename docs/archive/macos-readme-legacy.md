## macOS Notes

- Homebrew is strongly recommended for installing and maintaining developer tools.
- Modern macOS usually uses `zsh`; environment variables commonly live in `~/.zshrc` or `~/.zprofile`.
- Apple Silicon Homebrew is commonly installed under `/opt/homebrew`.
- Do not expose API keys when sharing screenshots or terminal output.
